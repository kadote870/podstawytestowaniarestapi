## Arkana szaleństwa w Java Script

```js
console.log(2 + 2)
// Wynik: 4

console.log("4" + 2)
// Wynik: 42
```

```js
console.log(4 - 2)
// Wynik: 2

console.log("4" - 2)
// Wynik: 2
```

```js
console.log(4 * 2)
// Wynik: 8

console.log("4" * 2)
// Wynik: 8
```

```js
console.log(4 / 2)
// Wynik: 2

console.log("4" / 2)
// Wynik: 2
```
